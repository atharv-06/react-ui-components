# 🚀 UI Components Library

A React + TypeScript + TailwindCSS project with **Storybook** documentation.  
Includes two reusable UI components: **InputField** and **DataTable**.

## 🛠 Tech Stack
- [React](https://react.dev/) (with TypeScript)
- [Vite](https://vitejs.dev/) (development & build tool)
- [TailwindCSS](https://tailwindcss.com/) (styling)
- [Storybook](https://storybook.js.org/) (component documentation)
- [Jest / React Testing Library](https://testing-library.com/) (basic tests)

## 📦 Installation

Clone this repo and install dependencies:

```bash
git clone <your-repo-url>
cd ui-components-starter
